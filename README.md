About
-----

This repository contains data and an IPython Notebook to reproduce some of the data analysis and plots in:

Geers AJ, Larrabide I, Radaelli AG, Bogunovic H, Kim M, Gratama van Andel HAF, Majoie CB, VanBavel E, Frangi AF. Patient-specific computational hemodynamics of intracranial aneurysms from 3D rotational angiography and CT angiography: An in vivo reproducibility study. Dependencies
------------

The notebook in this repository was successfully run with:
- [Python] 2.7
- [NumPy] 1.8
- [matplotlib] 1.3
- [VTK] 5.10

An easy way of installing these dependencies is to install [Anaconda]. Make sure to add VTK with `conda install vtk`.
